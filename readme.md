## About Cval

Conpensation management syatem

- capture user.
- capture finger print.
- print report.
- export to excel.
- print indemnity.


## Installation

```
git clone https://github.com/syflex/cval-backend.git
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan passport:install
```
